# Trading Platform

## Monorepo Structure
This repository follows a monorepo structure which contains the following key directories:

- **/backend**: Contains all server-side code and APIs.
- **/web**: Contains the web application code.
- **/mobile**: Contains the mobile application code.

## Backend Setup
To set up the backend, follow these steps:

1. Ensure you have Node.js installed on your machine.
2. Navigate to the `/backend` directory:
   ```bash
   cd backend
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Set up the environment variables in a `.env` file as per the `.env.example` provided.
5. Start the backend server:
   ```bash
   npm start
   ```
